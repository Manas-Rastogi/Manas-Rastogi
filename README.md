## 🚀 Featured Projects

### 🐾 [CarePaws — Stray Animal Emergency Response](https://github.com/Avenger-manas/CarePaws-Microservices-Project)
> Microservices platform | Spring Boot · RabbitMQ · Docker · OAuth2 · MongoDB

A real-time emergency response system that bridges citizens, NGOs, and municipal authorities. Citizens report injured stray animals; the system automatically routes alerts to nearby responders via event-driven messaging.

**Key highlights:** 6 independent microservices · JWT + RBAC security · Fully Dockerized · CI/CD via GitHub Actions

---

### 💼 [RecruitMate — Recruitment Management Backend](https://github.com/Avenger-manas/RecruitMate)
> REST API | Spring Boot · JWT · File Handling

Backend system for end-to-end recruitment: user auth, resume upload & parsing, job creation, and applicant tracking for admins.

---

### 🏥 [HealthTracker — Hemophilia Care Platform](https://github.com/Avenger-manas/HealthTracker-Backend)
> Healthcare API | Spring Boot · MongoDB

Connects hemophilia patients with nearby NGOs and hospitals. Features factor dosage calculation and emergency resource tracking.

---

### 🏛️ [CivicInsight — AI-Powered Political Office Command Center](https://github.com/Avenger-manas/CivicInsight-Backend)
> AI + Backend | Java · Ollama · Llama 3 · Geo-aware routing

Digitizes political offices with AI-driven grievance classification, location-aware issue tracking, and automated task assignment.
